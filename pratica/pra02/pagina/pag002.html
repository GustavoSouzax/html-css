Exercício de prática 02

Colocando links

Links externos e internos

Para criar hiperlinks utilizamos a tag a
Exemplo:
<p>Você pode acessar meu <a href="https://www.instagram.com/knights_works/">instagram</a> e me seguir lá.</p>

Nota: Na tag de hyperlink <a> tem como seu principal atributo o href.
Para poder controlar onde o site de destino vai abrir, podemos usar o atributo target, que suporta os seguintes valoraes.
_blank vai abrir o link em uma nova janela em branco
_self vai abrir o link na janela ou frame atual (padrão)
_top vai desfazer todos os frames e abrir o destino no navegador completo
_parent similar ao uso do _top em uma raferência a janela mãe nome-do-frame caso esteja usando frames, indicar o nome da jenla a abrir 

            <a href="pagina002.html" target="_self">
                Continuar navegando no site
            </a>
        
            <a href"https://www.     .com" target="_blank">
                Abrir link externo
            </a>
