Tags importantes

Tag de imagens

basta utilizar a tag img
Exemplo:
<img> src=”imagem.png” alt=”referencia da imagem”

Tag de hiperlink

Basta utilizar a tag a
Exemplo:
<a href=”url do hiperlink”>palavra que ficará clicavel</a>

Tag de favicon

Basta utilizar a tag link e selecionar link:favicon
Exemplo:
<link rel=”shortcut icon” href=”favicon.ico” type=”image/x-icon”>

Tag Negrito/ Destaque

Basta utilizar a tag b ou strong é aconselhavel sempre utilizar strong
Exemplo:
<p>Nesta frase, temos um <b>termo em negrito</b> usando a tag B(não semântica)</p>
<p>Nesta frase, temos um <strong>termo em destaque</strong> usando a tag STRONG(semântica)</p>

Tag Italico/ Ênfase

Basta utilizar a tag i ou em é aconselhavel sempre utilizar em
Exemplo:
<p>Nesta frase, temos um <i>termo em itálico </i>usando a tag I(não semântica)</p>
<p>Nesta frase, temos um <em>termo em ênfase</em> usando a tag E(semântica)M</p>

Tag marca texto

Basta utilizar a tag mark.
Exemplo:
<p>Podemos criar também um <mark>texto marcado usando</mark> a tag <mark></p>

Tag letras pequenas

Basta utilizar a tag small.
Exemplo:
<p>Estamos criando um <big>texto grande</big> e um <small>texto pequeno</small></p>

Nota: A tag big é obsoleta e não deve ser utilizada para mudar o tamanho da letra utilizamos as CSS. A tag small continua funcionando porque é uma tag semântica, como as letras pequenas de um contrato.

Tag texto deletado

Basta utilizar a tag del.
Exemplo:
<p>Podemos marcar um <del>texto como deletado</del> para indicar que ele deve ser lido, mas ignorado</p>

Nota: Antigamente se utilizava a tag strike para o mesmo propósito, mas com a atualização da HTML5 ela se foi substituida pela tag del.

Tag texto inserido

Basta utilizar a tag ins.
Exemplo:
<p>Podemos marcar um <ins>texto como inserido</ins> para dar ênfase e indicar que ele foi adicionado depois</p>

Nota: Antigamente se usava a tag u mas com a atualização da HTML essa tag se tornou obsoleta.

Tag texto sobrescrito

Basta utilizar a tag sup.
Exemplo:
<p>Para inserir coisas do tipo x<sup>20</sup>+3</p>

E assim ficará no documento aberto no navegador x20+3

Tag texto subrescrito

Basta utilizar a tag sub.
Exemplo:
<p>Para inserir coisas do tipo H<sub>2</sub>O</p>

E assim ficará no documento aberto no navegador H2O

Colocando emojis e caracteres especiais no site

Emojis

Para colocar emojis nos sites utilizamos &#x mais o código do emoji (disponível no emojipedia).

Exemplo:
<p>No final desta frase tem um emoji &#x1F60E;.</p>

Nota: O "X" presente no código é minúsculo, e sempre no final do código não esqueça de colocar ";".

Caracteres especiais

Para utilizar caracteres epeciais basta utilizar o & (e comercial).
Exemplo:
<p>Neste parágrafo temos o simbolo do euro &euro;</p>
